◆ abater_ativo()

void abater_ativo ( Ativos * ativos)

Abate um ativo (marca como ABATIDO) e regista a data.

Parameters
ativosApontador para a estrutura com a lista de ativos.

Abate um ativo (marca como ABATIDO) e regista a data.

Altera o estado de ativo para ABATIDO e utiliza a biblioteca time.h para capturar automaticamente o momento da operação. Impede que o faça se os ativos ja estiverem abatidos ou se estiverem a meio de uma manutenção.

Parameters
ativosApontador para a estrutura que contém a lista de ativos.
Note
o Ativo permanece no programa, só deixará de ser válido em funções como procurar_ativo_id().
